# Magic 8 Ball Web Application 🎯

### Project Description
This project brings the classic Magic 8 Ball experience to the web! With a simple interface and a responsive ball, users can ask any yes/no question and receive a mystical answer.

### The Problem (that doesn't exist)
In today’s tech-driven world, everyone’s decisions are powered by data, facts, and logic. But what about fun, whimsical, and unpredictable answers to life’s questions? Where’s the magic when we need it?

### The Solution (that nobody asked for)
The problem of missing mystical answers is solved by creating an online Magic 8 Ball! Just ask your question, and the ball will rotate, making decisions for you, complete with sound effects and animations, just like in the movies. Magic has never been this interactive!

## Technical Details
### Technologies/Components Used
For Software:

* Languages used: HTML, CSS, Python (with Flask backend)
* Frameworks used: Flask (Python framework)
* Libraries used: Fetch API, Web Audio API (for sound)
* Tools used: Flask Development Server, Chrome DevTools for debugging
